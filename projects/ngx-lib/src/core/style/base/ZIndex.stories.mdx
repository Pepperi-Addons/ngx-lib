# Z Index

The see the z-index play out in the box below

- `$z-index-high`
- `$z-index-medium`
- `$z-index-low`
- `$z-index-lowest`


Use them like so `↓`

```css
// Make sure you call the `variables.scss` at the top of the file
@use "~/src/core/style/abstracts/variables";

.high-class {
    z-index: variables.$z-index-high;
}
```
